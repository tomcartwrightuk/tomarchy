# Tomarchy

A bootstrap script for setting up Arch Linux with a curated selection of tools and visual customisations.

Heavily influenced by [Omarchy](https://github.com/basecamp/omakub).

## Features

- **One-liner install** — Fetch and run via `curl` on a fresh Arch installation
- **System updates** — Upgrades base packages
- **Hyprland** — Installs Hyprland and its dependencies for a modern Wayland compositor experience
- **Git** — Ensures Git is installed and ready to go
- **WiFi support** — Configures networking so you can connect wirelessly

## Installation

```bash
curl -fsSL https://raw.githubusercontent.com/tomcartwrightuk/tomarchy/main/install.sh | bash
```

## Requirements

- A fresh Arch Linux installation
- Internet connection (ethernet recommended for initial setup) or you can use iwtcl.
